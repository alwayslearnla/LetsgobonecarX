Adicionar boneca nova

Preencha os dados abaixo e gere o trecho prontinho para colar no produtos.json.

1 Suba as fotos primeiro

No GitHub, abra a pasta fotos → Add file → Upload files → arraste as fotos da boneca → Commit changes. Dê nomes simples, sem espaço ou acento (ex: 003-a.jpg). Depois volte aqui e preencha os campos abaixo.

2 Preencha os dados da boneca

Preencha pelo menos o nome e uma foto antes de gerar.

3 Copie e cole no produtos.json

Abra o produtos.json no GitHub (ícone de lápis), clique logo depois do } do último bloco, digite uma vírgula, cole o trecho abaixo, e clique em Commit changes.
